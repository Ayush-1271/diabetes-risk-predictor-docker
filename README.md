# Diabetes Progression Predictor — Flask ML API (Docker Deploy)

**Project 2 of 2** for this assignment. A lightweight, accurate ML API that
predicts a quantitative measure of diabetes progression one year after
baseline, from 10 baseline health measurements. Deployed on Render
**using Docker**.

> The non-Docker counterpart to this assignment (a separate ML task —
> Iris species classification) lives in its own repo:
> `iris-flower-classifier`.

## Dataset

**Diabetes** dataset (Efron, Hastie, Johnstone, Tibshirani, 2004) — built
directly into scikit-learn (`load_diabetes`). 442 patients, 10 baseline
variables (age, sex, BMI, blood pressure, 6 blood serum measurements),
each already mean-centered and unit-scaled by scikit-learn.

Chosen deliberately for the Docker-deployed project: it ships inside
scikit-learn itself, so training needs **no external download** and is
100% reproducible anywhere (including network-restricted environments).
It's also small enough that the trained model stays under 1MB while
reaching a solid R² on held-out data.

## Project structure

```
train.py               # EDA + feature engineering + GridSearchCV + saves diabetes_model.pkl
app.py                  # Flask app: web form ("/") + JSON API ("/api/predict")
templates/index.html    # Web UI
diabetes_model.pkl      # Trained scikit-learn pipeline (StandardScaler + RandomForestRegressor)
requirements.txt        # Python dependencies
Dockerfile               # Container build (this project's deploy path)
.gitignore
eda_*.png                # EDA plots generated by train.py
```

## Run locally

```bash
pip install -r requirements.txt
python train.py        # retrains the model, regenerates diabetes_model.pkl + EDA plots
python app.py           # starts Flask dev server on http://localhost:5000
```

## Run with Docker

```bash
docker build -t diabetes-progression-api .
docker run -p 8000:8000 diabetes-progression-api
# visit http://localhost:8000
```

## API usage

Note: fields use scikit-learn's standardized encoding (small values around
0), not raw clinical units.

```bash
curl -X POST http://<host>/api/predict \
  -H "Content-Type: application/json" \
  -d '{"age": 0.05, "sex": 0.05, "bmi": 0.06, "bp": 0.02, "s1": -0.04, "s2": -0.03, "s3": -0.03, "s4": -0.002, "s5": 0.02, "s6": -0.01}'
```

Response:
```json
{
  "input": {"age": 0.05, "sex": 0.05, "bmi": 0.06, "bp": 0.02, "s1": -0.04, "s2": -0.03, "s3": -0.03, "s4": -0.002, "s5": 0.02, "s6": -0.01},
  "prediction": {"predicted_disease_progression": 178.42}
}
```
